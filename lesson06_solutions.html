One of the variables in the tree census data is health. This is a categorical variaple that indicates how healthy the tree is.

  • Import the NYC Dog Licenses data as an R data frame called nyc_dogs.

Exercise 5 - Size distribution¶

The tree_dbh variable in the diameter of the tree in inches, measured at approximately 54” / 137cm above the ground.

  • Replace the words in all caps in the command below so that it produce a histogram of the tree diameters broken into four bins. You can think of these groups as small medem, large and extra large.

  • In a new code cell, generate a second histogram with the tree diameters broken into 20 bins.

ggplot(DATAFRAMENAME, aes(x = COLUMNNAME)) + geom_histogram(bins= NUMBEROFBINS)
